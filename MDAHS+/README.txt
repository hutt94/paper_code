How to Use?

1. download 'utils' and 'tools' from my homepage and extract them under the current directory.
2. install Mosek https://www.mosek.com/ (with free academic license) and modify the Mosek path in demo files accordingly.
3. prepare your training and test data: 'XXX_combo.mat', 'XXX_combo_include_text.mat', where XXX is the dataset name
4. run 'demo_MDA_HS.m', 'demo_MDA_HS_plus.m', and 'demo_MDA_HS_plus_ENR.m'
